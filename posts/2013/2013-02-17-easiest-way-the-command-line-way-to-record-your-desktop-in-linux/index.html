Easiest way (the command line way) to record your desktop (screencast) in linux

Use this script. Make sure ~/Video directory exits on your computer. And ffmpeg is also installed. This script computes the resolution of your desktop and saves the video to ~/Video/output.mkv file. The file size is typically very big (high definition) which you can again reduce to by converting the file to some other format using ffmpeg. No need to use GUI application.
